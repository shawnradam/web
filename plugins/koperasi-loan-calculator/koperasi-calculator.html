Standalone Web Plugin

Koperasi Personal Loan Calculator

A dependency-free, modern, highly configurable loan widget with real-time upfront fee deductions, SVG chart visualization, and amortization schedule.

⚡
Real-time Upfront Deductions
Accurately subtracts processing fees, Takaful insurance, membership fees, and advance payments from the principal to calculate the net payout.
📊
SVG Donut Breakdown
Visual real-time SVG Donut chart displaying payout vs interest cost vs deductions without any heavy charting library.
🛠️
Live Configurator
Modify interest rates, fee percentages, and settings inline, and copy the dynamic JSON initialization code immediately.

Integration Code Templates

<!-- 1. Include the styles inside <head> --> <link rel="stylesheet" href="plugins/koperasi-loan-calculator/koperasi-calculator.css"> <!-- 2. Put target element --> <div id="my-calculator-widget"></div> <!-- 3. Include script and initialize --> <script src="plugins/koperasi-loan-calculator/koperasi-calculator.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { KoperasiCalculator.init('my-calculator-widget', { theme: 'cosmic', // Options: 'light', 'dark', 'cosmic' defaultLoan: 30000, defaultTenure: 5, calculationMode: 'flat' // Options: 'flat', 'reducing' }); }); </script>
<?php // 1. Require the plugin helper class require_once 'plugins/koperasi-loan-calculator/koperasi-calculator.php'; // 2. Output the styles and scripts in your <head> or layout KoperasiLoanCalculator::register_assets(); // 3. Render the widget inside your container echo KoperasiLoanCalculator::render('my-php-calculator', [ 'theme' => 'cosmic', 'defaultLoan' => 45000, 'defaultTenure' => 6 ]); ?>
